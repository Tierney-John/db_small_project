## 项目名称

DIY2048
    
### 作者

Tierney-John
    
### 项目简介

这是一款WebAPP,是一个纯净版的2048小游戏，它在PC和PE端的浏览器上都可以正常运行。
    
### 版本日志

* v1.0.0    正式发行版

### 待办事项

PE端会与浏览器存在些许不兼容，部分浏览器会影响游戏的正常运行
建议iPhone用户使用Safari浏览器，Android用户使用Google浏览器或115浏览器
对于以上问题，将在下一个版本中进行修正
